- Loading the Cell2Cell dataset
- Selecting numerical features
- Converting rows into vectors
- Calculating Euclidean distance manually
- Understanding how KNN compares examples using distance

## Main Concepts

KNN is an instance-based supervised learning algorithm. Instead of building a complex model during training, it stores the training examples and uses distance to find the nearest neighbors when making predictions.

Important concepts practiced:
- K-nearest neighbors
- Euclidean distance
- Feature vectors
- Train/test split
- Classification accuracy
- Curse of dimensionality
- Feature scaling
